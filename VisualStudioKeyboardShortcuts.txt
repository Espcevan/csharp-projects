Visual Studio shortcuts:
  Format code: Select code then - Ctrl + k, Ctrl + f
  Format whole file: ctrl + k, ctrl + d
  Move between open tabs: ctrl + f6
  Move between open tabs backwards: ctrl + shift + f6
  Search data: ctrl + ,
  Search solution explorer: ctrl + ;   Select stuff with arrowKeys then enter
  Search everything: ctrl + t
  Cycle Through open tabs forward: ctrl + tab
  Cycle Through open tabs backward: ctrl + shift + tab
  Quick collapse a method, class, ect: put cursor where you want collapsed and press ctrl + m(twice)
  Build project without running: ctrl+shift+b
  Comment out selected lines: ctrl k + ctrl c
  UnComment out selected lines: ctrl k + ctrl u
  Rename selected variable/method/ect: ctrl r + ctrl r
  Changes selected text to lowercase: ctrl + u
  Surround with: ctrl k + ctrl s
  Quick actions for current text under cursor: ctrl + .
  Run with debugger: f5
  Run without debugger: ctrl + f5
  Stop debugging: Shift + f5
  Close current file: ctrl + f4
  Inline Definition peek: alt + f12
  Class View: ctrl + shift + c
  F4: Properties windows for selected item
  Object browser: ctrl+alt+j
  Find all references: shift + f12
  Show method parameters: ctrl + shift + space
  Put blank line under cursor: ctrl + shift + enter
  Incremental search: ctrl + i 
  Find in files: ctrl + shift + f
  Cut current line: ctrl + L
  Delete current line: ctrl + shift + L
  Place bookmark: ctrl + k, ctrl + k
  Goto next bookmark: ctrl + k, ctrl + n
  Jump to ending brace: ctrl + ]
  Lists members for statement completion: Ctrl + J	
  
  Website with all shortcut listings: http://www.dofactory.com/reference/visual-studio-shortcuts
